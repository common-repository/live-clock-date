== Description ==

Live Clock Date Widget is a kind of free plugins for wordpress-based blogs. This Plugin presents the combination between date and clock, whereas clock on this plugin runs real time. The combination can be applied on this plugin is in the form of settings between two options, displaying the calendar first or vice versa. And also, the calendar on this plugin can be activated and only displays real time clock. Besides, there is also available Style and Design settings that is simple and easy to understand. 

This plugin is adjustable to the themes that is using responsive system.

Features Offered on this plugin are:

* Displaying the calendar and Real Time clock combination
* Flexible settings  for Calendar and Realtime Clock combination
* Basic Settings for Style and Design
* Easy-to-understand settings 
* Applicable for responsive system themes 

Features mentioned above will keep being developed and the functions also keep being improved adjusted to the development and the need.

Hopefully this plugin gives benefits for all of us.

== Installation ==

Upload the Live Clock Date plugin to your blog, Activate it, then go to Configure Widget.

1, 2, 3: You're done!

Then Enable this plugin to your sidebar, choose your setting

4, 5: Plugin publish now
